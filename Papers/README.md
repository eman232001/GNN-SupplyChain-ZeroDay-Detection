# Research Papers Collection

This directory contains all research papers reviewed during my Master Thesis research.

The papers focus on:

- Graph Neural Networks (GNN) for security analysis
- Software vulnerability detection
- Software Supply Chain Security
- Zero-day and unknown vulnerability prediction
- Graph-based machine learning approaches

Each paper folder includes:

- Research paper (PDF if permitted)
- Paper summary and key contributions
- Dataset and methodology notes
- Important observations for literature survey comparison
- Relation to the proposed research model

These papers form the foundation of the Literature Review and Survey analysis of this research.
